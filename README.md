Task 2 – AI Content Marketing using UGC Ads

This project focuses on creating AI-generated UGC-style advertisement
content for a local gym using structured prompt engineering.

Business:
- FitZone Gym
- Location: Belagavi

Tools Used:
- ChatGPT
- Google Docs
- GitHub

Deliverables:
- Scroll-stopping hooks
- UGC-style ad scripts following Problem → Solution → CTA format
- Short CTA lines for conversions
- Social media captions for Instagram, Facebook, and YouTube Shorts

This task demonstrates the use of AI-driven content marketing,
brand voice adaptation, and conversion-focused UGC ad creation
for a real-world fitness business.
